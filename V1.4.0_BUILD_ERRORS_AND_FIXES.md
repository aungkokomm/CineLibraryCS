## 🔧 Key API Reference (For Fixes)

**AppState Usage**:
```csharp
var db = AppState.Instance.Db;  // ✅ CORRECT
var db = AppState.Services.GetRequiredService<DatabaseService>();  // ❌ WRONG
```

**Database Methods**:
```csharp
db.GetMovies(options)  // Returns List<MovieListItem>
db.GetTotalRuntimeHours()  // Returns double
db.GetWatchProgress()  // Returns (int watched, int total, double percentage)
db.GetMoviesByDecade()  // Returns List<(int decade, int count, double avgRating)>
db.GetTopDirectors(limit)  // Returns List<GenreFacet>
db.GetTopActors(limit)  // Returns List<GenreFacet>
```

**Window vs ContentDialog**:
```csharp
// MovieDetailDialog is a Window
var dialog = new MovieDetailDialog(movieId);
dialog.Activate();  // Show it, NOT ShowAsync()

// KeyboardShortcutsDialog is a ContentDialog
var dialog = new KeyboardShortcutsDialog();
await dialog.ShowAsync();  // Show it with await
```
