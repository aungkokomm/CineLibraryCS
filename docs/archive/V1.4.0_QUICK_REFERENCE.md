## 📋 Architecture Reference

### Database Service Pattern
```csharp
var db = AppState.Services.GetRequiredService<DatabaseService>();
var data = db.GetSomeData();
```

### ViewModel Pattern
```csharp
public class MyViewModel : ObservableObject
{
    [ObservableProperty]
    private string myProperty;

    public void MyMethod() { }
}
```

### Navigation Pattern
```csharp
ContentFrame.Navigate(typeof(Views.SomePage));
SidebarTitle.Text = "Page Title";
```

### Data Binding Pattern
```xaml
<TextBlock Text="{x:Bind ViewModel.SomeProperty}"/>
<ListView ItemsSource="{x:Bind MyCollection}">
    <ListView.ItemTemplate>
        <DataTemplate x:DataType="local:ItemType">
            <TextBlock Text="{x:Bind Name}"/>
        </DataTemplate>
    </ListView.ItemTemplate>
</ListView>
```

## 🐛 Common Issues & Fixes

### Issue: XAML Parser Error
**Fix**: Check `x:Class` namespace matches file location. Make sure all XAML elements are closed properly.

### Issue: "InitializeComponent not found"
**Fix**: Make sure code-behind class name matches XAML `x:Class` attribute.

### Issue: Data binding not working
**Fix**: Check `DataContext` is set. For `x:Bind`, make sure property exists and is public.

### Issue: Navigation fails
**Fix**: Make sure page class exists and namespace is correct in `Navigate(typeof(...))`.

### Issue: Database returns null/empty
**Fix**: Check database file exists and has data. Test query directly.
