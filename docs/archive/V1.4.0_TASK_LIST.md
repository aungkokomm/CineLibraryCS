### Task 1.2: Create StatisticsPage.xaml.cs CodeBehind
**Difficulty**: ⭐⭐ (Medium)  
**Time**: 1-2 hours  
**What**: Implement the view logic and data binding

**Checklist**:
- [ ] Create `Views/StatisticsPage.xaml.cs`
- [ ] Create `StatisticsViewModel` class (or use existing LibraryViewModel)
- [ ] Add properties for each statistic:
  - [ ] `ObservableCollection<(int Decade, int Count)> DecadeStats`
  - [ ] `ObservableCollection<GenreFacet> TopDirectors`
  - [ ] `ObservableCollection<GenreFacet> TopActors`
  - [ ] `string TotalRuntimeHours`
  - [ ] `string WatchProgress`
- [ ] Implement LoadStatistics() method
- [ ] Add refresh button click handler
- [ ] Call database methods via DatabaseService

**Code Skeleton**:
```csharp
public sealed partial class StatisticsPage : Page
{
    private readonly DatabaseService _db;
    public ObservableCollection<(int, int)> DecadeStats { get; set; }
    public ObservableCollection<GenreFacet> TopDirectors { get; set; }
    // ... other properties

    public StatisticsPage()
    {
        this.InitializeComponent();
        _db = AppState.Services.GetRequiredService<DatabaseService>();
    }

    private async void OnLoaded(object sender, RoutedEventArgs e)
    {
        await LoadStatisticsAsync();
    }

    private async Task LoadStatisticsAsync()
    {
        DecadeStats.Clear();
        foreach (var (decade, count) in _db.GetMoviesByDecade())
        {
            DecadeStats.Add((decade, count));
        }
        // Similar for TopDirectors, TopActors, etc.
    }

    private void OnRefresh(object sender, RoutedEventArgs e)
    {
        _ = LoadStatisticsAsync();
    }
}
```

**Testing**:
- [ ] Page loads without errors

                <HyperlinkButton Content="{Binding}"
                                 Click="OnDirectorClick"
                                 Tag="{Binding}"/>
            </ItemsControl.ItemTemplate>

    </TextBlock>

```csharp
// In MovieDetailDialog.xaml.cs
private void OnDirectorClick(object sender, RoutedEventArgs e)
{
    var directorName = (sender as HyperlinkButton)?.Tag as string;
    if (!string.IsNullOrEmpty(directorName))
    {
        var libraryPage = (Window.Current.Content as MainWindow)?.ContentFrame
            .Content as LibraryPage;
        libraryPage?.ViewModel.FilterByDirector(directorName);

        this.Close();
    }
}
```
