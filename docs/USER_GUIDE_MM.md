# CineLibrary — လက်စွဲလမ်းညွှန်

> Windows အတွက် native ရုပ်ရှင် Library။
> Hard Drive, SSD အလုံးအရေ အကန့်အသတ်မရှိ ကြည့်ရှု၊ ရှာဖွေ၊ မှတ်တမ်းတင်နိုင်သည်။

ဤလက်စွဲသည် သင့်အား အသစ်တပ်ဆင်ချိန်မှ ငါးမိနစ်အတွင်း ထောင်ချီသော ရုပ်ရှင် Catalog တစ်ခု ဖန်တီးနိုင်သည်အထိ လမ်းညွှန် တွဲခေါ်ပေးပါလိမ့်မည်။

> **v2.9.0 အတွင်း ဆန်းသစ်ထားမှုများ**
> - **🟣 TV အတွက် Continue Watching** — စကြည့်ထားပြီး မပြီးဆုံးသေးသော Show များအတွက် နောက်ထပ်ကြည့်မယ့် episode ကို ထိပ်တွင်ပြသပေးပါသည်။
> - **🕓 Recently Watched** — နောက်ဆုံး play ထားသည့် movie များကို sidebar မှ တစ်ချက်တည်းနှင့် ပြန်ရှာနိုင်ပါသည်။
> - **📅 On This Day** — ယနေ့ရက်စွဲ၌ ယခင်နှစ်များက ကြည့်ခဲ့သော ရုပ်ရှင်များ၊ ထွက်ရှိခဲ့သော ရုပ်ရှင်များကို ပြသပါသည်။
> - **🏷 Tags** — ရုပ်ရှင်နှင့် Show များကို "rewatched", "Christmas" စသည် ကိုယ်ပိုင် label များဖြင့် တွဲမှတ်နိုင်ပါသည်။
> - **★ Episode အလိုက် Favorite နှင့် Note** — episode တစ်ခုစီကို မှတ်သားနိုင်ပါပြီ။
> - **🎲 Filter-aware Surprise Me** — ဘာကြည့်ရမလဲ ရွေးရခက်နေသည့်အချိန်တိုင်းအတွက် လက်ရှိ filter အတိုင်း ကျပန်းရွေးပေးနိုင်သော 🎲 ခလုတ်။
> - **🔍 Search တွင် TV Shows** — search box သည် ရုပ်ရှင်နှင့်အတူ TV Show များကိုပါ ရှာပေးနိုင်ပါပြီ။
> - **💾 Backup / Restore** — ကိုယ်ပိုင်ဒေတာ အားလုံးကို JSON file တစ်ခုထဲ၌ export လုပ်နိုင်/import လုပ်နိုင်သည်။
> - **📤 List ကို image အဖြစ် Export** — list တစ်ခုကို poster grid PNG အဖြစ် ထုတ်ယူနိုင်သည်။
> - **⌨️ Keyboard shortcuts တိုးချဲ့** — `/` search၊ `F` favorite၊ `W` watchlist၊ `Delete` list မှ ဖယ်ရှား။
> - **⚡ စွမ်းဆောင်ရည် တိုးတက်** — App တခုလုံး ပိုမို သွက်လက်မြန်ဆန်အောင် code quality နှင့် performance pass စစ်ဆေးပြုပြင်ထားပါသည်။
>

> cast, season-by-season episode row များ၊ episode တစ်ခုစီ watched
> မှတ်တမ်း၊ "▶ Play next unwatched" ခလုတ်)။ *TV Shows* အောက်တွင် ကြည့်ပါ။

## CineLibrary စတင်အသုံးမပြုသေးခင် စနစ်တကျ ပြင်ဆင်ပါ

CineLibrary သည် **ဖတ်ရုံသာ** ဖြစ်သည် — Scraper မဟုတ်ပါ။ 
သင့်ရုပ်ရှင်ဖိုင်တွဲများကို [MediaElch](https://www.kvibes.de/mediaelch/) ဖြင့် ကြိုတင် Scrape ပြုလုပ်ထားရပါမည်။ 
ရုပ်ရှင်တစ်ကားစီ မိမိဖိုင်တွဲထဲတွင် `.nfo` ဖိုင်နှင့် poster ပုံများ ရှိနှင့်ပြီးသား ဖြစ်ရပါမည်။ 

**ရှိနှင့်ပြီးသား ဖြစ်ရမည့် ပုံစံ —**

```
H:\Movies\
├── Inception (2010)\
│   ├── Inception.mkv
│   ├── Inception.nfo
│   ├── poster.jpg
│   └── fanart.jpg
├── The Matrix (1999)\
│   ├── The Matrix.mp4
│   ├── The Matrix.nfo
│   └── poster.jpg
└── ...
```

**သင့် မူဗီဖိုင်များ အထက်ပါအတိုင်း ဖိုင်နာမည် မတည့်၊ မူဗီတခု ဖိုင်တခု အစီအစဉ်တကျ မရှိနေပါက 
သင့် Movie file နာမည်များ တည့်မတ်စေရန်နှင့် Folder📂 တခုချင်းစီတွင် အထာတကျ စီပေးရန်
[CineLibraryEssentials](https://github.com/aungkokomm/CineLibraryEssentials)
ကို အသုံးပြု ရှင်းလင်းပြင်ဆင်ပြီးမှ (သို့မဟုတ်) **MediaElch** ဖြင့် အရင် Scrape ပြုလုပ် ပြီးမှသာ...
`.nfo` ဖိုင်များ ရှိမှသာ CineLibrary ကိုအသုံးပြုပါရန် အကြံပြုပါသည်။**

လိုအပ်ချက်များ —

- Windows 10 (build 19041) သို့မဟုတ် Windows 11
- app အတွက် ~150 MB နေရာ၊ ရုပ်ရှင် 10,000 ကားပါ catalog ~50 MB

## Install ပြုလုပ်ခြင်းနှင့် ပထမဆုံးဖွင့်ခြင်း

1. [releases page](https://github.com/aungkokomm/CineLibraryCS/releases/latest) မှ နောက်ဆုံး installer ဒေါင်းလုဒ် ပြုလုပ်ပါ — `CineLibrary-vX.Y.Z-Portable-Setup.exe`။
2. Installer ဖွင့်ပါ။ **Portable** installer ဖြစ်သောကြောင့် သင်ကြိုက်ရာ ဖိုင်တွဲတွင် ထည့်သွင်းနိုင်သည် — Documents, external drive, USB stick မည်သည့်နေရာမဆို။ App နှင့် database ပူးတွဲ တည်ဆောက်ပါလိမ့်မည်။
3. `CineLibrary.exe` ဖွင့်ပါ။ ရုပ်ရှင်ဖိုင်တွဲ မထည့်မချင်း ပထမ window ဗလာဖြစ်မည်။

> **Portable ဆိုသည်မှာ?** Library ဒေတာများ `CineLibrary-Data\` ဖိုင်တွဲတွင် exe နှင့် အတူပူးတွဲ ရှိနေပါမည်။ ဖိုင်တွဲ တစ်ခုလုံး ရွှေ့ပြောင်း၍ ကွန်ပျူတာအသစ်တွင်လည်း ချက်ချင်းအသုံးပြုနိုင်သည်။

## ရုပ်ရှင်ဖိုင်တွဲများ ထည့်သွင်းခြင်း

Sidebar မှ **Drives** စာမျက်နှာ ဖွင့်ပါ။

**Add folder** နှိပ်၍ ရုပ်ရှင်များရှိသော root ဖိုင်တွဲ (ဥပမာ `H:\Movies` သို့မဟုတ် `E:\Films`) ရွေးပါ။ 
CineLibrary သည် သင်ညွှန်းသည့် Drieve/Folder သို့သွား၍ `.nfo` ဖိုင်တစ်ခုစီ ဖတ်ကာ poster များ cache ပြုလုပ်ပြီး catalog တည်ဆောက်ပါလိမ့်မည်။ တိုးတက်မှုကို window အောက်ဘယ်ဘက်တွင် toast ဖြင့် ပြသမည်။

အကြံများ —

- **Root ဖိုင်တွဲ မည်မျှမဆို ထည့်နိုင်သည်။** External drive တစ်ခုစီ သီးခြားထည့်နိုင်သည်။
- **Drive အမည်ပြင်ဆင်ပါ။** Drive entry ကို right-click နှိပ်၍ အမည်ပြောင်းနိုင်သည် (ဥပမာ *"Seagate Red 5TB — ရုပ်ရှင်များ"*)။
- **MediaElch ဖြင့် ပြန် scrape ပြုလုပ်ပြီးနောက်** drive entry ပေါ်တွင် **Rescan** နှိပ်ပါ။ ဖိုင်အသစ်နှင့် ပြောင်းလဲမှုများ CineLibrary ကောက်ယူမည်။
- **Drive ဖျက်ခြင်း** — သင့်ရုပ်ရှင်ဖိုင်များ မပျက်ပါ၊ CineLibrary ၏ catalog မှတ်တမ်းထဲမှသာ ပယ်ဖျက်သည်။ မူလ Drive ထဲရှိ ဖိုင်များ ပတကိအတိုင်း ရှိနေပါမည်။ 

Scan ပြီးနောက် Sidebar မှ **All Movies** နှိပ်ပါ။

## Library ကြည့်ရှုခြင်း

**All Movies** စာမျက်နှာသည် ပင်မ view ဖြစ်သည်။ Sidebar တွင် catalog ကို သက်ဆိုင်ရာ section အလိုက် ပြထားပါသည်။

### Grid view (မူလ)

Poster များ fluid grid ပုံစံ ဖော်ပြသည်။ Window ကြီးသည်နှင့် column အရွယ်အစားကို အလိုအလျောက် ချိန်ညှိပါလိမ့်မည်။ ညာဘက်ထိပ်ရှိ **S / M / L / XL** ရွေးချယ်မှုဖြင့်လည်း poster အရွယ်အစား ချိန်ညှိနိုင်သည်။

Poster တစ်ခု **တစ်ချက်နှိပ်** လျှင် **ရုပ်ရှင် အသေးစိတ်** dialog ပွင့်ပါမည် — အပြည့်ဖွင့်ပြီး cast၊ rating များ (IMDb / TMDb / Rotten Tomatoes)၊ tech badge များ (resolution, codec, audio)၊ file size နှင့် duration၊ ကြည့်ပြီး/မကြည့်ရသေး toggle များ ပြသထားပါမည်။ ပိတ်ရန် **Esc** ကိုနှိပ်ပါ။

Poster ကို **နှစ်ချက်နှိပ်** လျှင် မူလ video player ဖြင့် ချက်ချင်း
ဖွင့်ပါမည်။ ရုပ်ရှင်ရှိသော drive မချိတ်ဆက်ထားလျှင် မည်သည့် drive ကို
ချိတ်ဆက်ရမည်ကို CineLibrary က ပြောပြပါမည်။

### List view

Density picker ဘေးရှိ **☰** toggle နှိပ်လျှင် compact list view ပြောင်းသည် — ခေါင်းစဉ်၊ နှစ်၊ rating၊ runtime၊ ကြည့်ပြီးမပြီး ပြသည်။

### Sidebar ဖြတ်လမ်းများ

- **LIBRARY** — All Movies, Favorites, To Watch
- **DISCOVER** — Continue Watching, Recently Added, Surprise me
- **BROWSE** — *By Genre*, *By Decade*, *By Rating*, *Collections* —
  banner ပုံစံ စာမျက်နှာများ၊ category တစ်ခုနှိပ်လျှင် library ကို
  ထို category ဖြင့် filter ပြုလုပ်သည်
- **TOOLS** — Statistics, Drives
- **MY LISTS** — ကိုယ်ပိုင် list ဖန်တီးနိုင်သည် (**+** ဖြင့်)
- **LIBRARIES** — Drive root တစ်ခုစီ၊ အွန်လိုင်း/အော့ဖ်လိုင်း dot နှင့် ရုပ်ရှင်အရေအတွက်

Header တစ်ခုစီကို နှိပ်၍ ချုံ့/ချဲ့နိုင်သည်။

## ရှာဖွေ၊ စစ်ထုတ်နှင့် စီစစ်ခြင်း

### ရှာဖွေခြင်း

ထိပ်ဆုံးရှိ search box သည် **ခေါင်းစဉ်၊ သရုပ်ဆောင်၊ ဒါရိုက်တာ၊ plot** တို့ကို တစ်ပြိုင်နက် ရှာသည်။ ရိုက်သည်နှင့်အမျှ ရလဒ် တဆင့်ချင်း ပြောင်းမည်။ `Ctrl+F` နှိပ်လျှင် cursor ချက်ချင်း search box သို့ ရောက်သည်၊ `Esc` နှိပ်လျှင် ရှင်းသည်။

### Filter pill များ

ခေါင်းစဉ်အောက်ရှိ pill သုံးခုဖြင့် ကြည့်ပြီး/မကြည့်ရသေး စစ်ထုတ်သည် —

- **All** — ရုပ်ရှင်အားလုံး
- **Unwatched** — မကြည့်ရသေးသောများ
- **Watched** — ကြည့်ပြီးသောများ

Sidebar ဖြတ်လမ်းနှင့် ပေါင်းသုံးနိုင်သည် (ဥပမာ *"Drama genre ထဲမှ မကြည့်ရသေးသောများ"*)။

### စိစစ်ခြင်း

Sort dropdown တွင် ခေါင်းစဉ်၊ နှစ်၊ rating၊ runtime၊ ထည့်သွင်းချိန် — တက်/ဆင်း နှစ်မျိုးစီ ပါသည်။ နောက်ဆုံးရွေးချယ်မှု session များကြားတွင် မှတ်ထားသည်။

## ကြည့်ပြီး/မကြည့်ရသေး မှတ်တမ်းတင်ခြင်း

ရုပ်ရှင် အသေးစိတ် dialog မှ toggle နိုင်သော ရွေးချယ်စရာ သုံးခု —

- **Watched ✓** — ကြည့်ပြီးဟု မှတ်သည်။ Watched / Unwatched filter နှင့် Statistics စာမျက်နှာတွင် ပြပေးထားပါမည်။
- **Favorite ⭐** — Sidebar ၏ **Favorites** view တွင် ပြပေးထားပါမည်။
- **Watchlist 📋** — နောက်မှကြည့်မည်ဟု စီစဉ်သည်။ Sidebar ၏ **To Watch** တွင် badge counter ဖြင့် ပြပေးထားပါမည်။

ဤ flag များသည် **CineLibrary** တွင်သာ သိမ်းသည် — `.nfo` ဖိုင်ထဲ မရေးပါ။ 
MediaElch ဖြင့် ပြန် scrape လုပ်ခဲ့သည်ရှိသော် မထိခိုက်စေရန် ကြိုတင်ကာကွယ်ထားခြင်း ဖြစ်ပါသည်။

## TV Shows

**v2.8.0** မှစ၍ CineLibrary သည် ရုပ်ရှင်များနှင့်အတူ TV Show များကိုပါ
catalog ပြုလုပ်ပေးပါသည်။

### TV Show အဖြစ် ဘယ်လို မှတ်သလဲ

**TV Show များသည် Movie နှင့် သဘောသဘာဝ အင်မတန် ကွာခြားသည့်အတွက်...
Show တခုအား `tvshow.nfo`** ဖြင့် မှတ်သားလေ့ ရှိပါသည်။ 
စနစ်တကျ Scrape လုပ်ထားပါက folder တိုင်းတွင် `tvshow.nfo` ပါဝင်ပါမည်။
`tvshow.nfo` ပါဝင်သော Folder တိုင်းကို TV Show အဖြစ် သတ်မှတ်ပါသည်။ 
၎င်း folder ထဲရှိ `SxxExx` ပါသော video ဖိုင်များ (matching `.nfo` နှင့်) ကို
episode အဖြစ် ဖတ်သည် — **MediaElch** (သို့မဟုတ်) **CineLibrary Essentials** တို့မှ ထုတ်ပေးသော
ပုံစံအတိုင်း —

```
H:\TV\Dark\
├── tvshow.nfo                       ← show metadata
├── poster.jpg  fanart.jpg
├── .actors\                         ← cast ဓာတ်ပုံများ
├── Dark - S01E01 - Secrets.mkv
├── Dark - S01E01 - Secrets.nfo
├── Dark - S01E01 - Secrets-thumb.jpg
└── …
```

Season ကို ဖိုင်နာမည်ထဲက `Sxx` မှ တိုက်ရိုက်ဖတ်သည် — Season အားလုံးကို
folder တစ်ခုထဲ ပြားပြားထားနိုင်သည်။ ရုပ်ရှင်များ မထိခိုက်ပါ — Drive
တစ်ခုတည်းတွင် ရုပ်ရှင်ရော TV Show ရော ထားနိုင်သည်။

### TV Show များ ရှာဖွေခြင်း

Sidebar (All Movies အောက်) တွင် **All TV Shows** နှိပ်ပါ။ Show card grid
ပေါ်လာမည် — poster, နှစ်, watched progress bar တို့ဖြင့်။ Sidebar badge
က Show အရေအတွက် ပြသည်။

### Show စာမျက်နှာ

Show တစ်ခုနှိပ်လျှင် ၎င်း၏ စာမျက်နှာ ဖွင့်သည် — drilling မလို၊ scroll
တစ်ခုတည်းတွင် —

- **Header** — poster, plot, နှစ် · rating · status, watched roll-up
  ("12/62 watched")၊ genre chips, **cast strip**, **☆ Favorite** /
  **📋 Watchlist** ခလုတ်များ။
- **▶ Play next** — မကြည့်ရသေးသော ပထမ episode သို့ ခုန်သည် (season,
  episode အလိုက်)။ ပြီးသွားလျှင် "✓ All watched" ပြသည်။
- **Season row များ** — Season တစ်ခုစီသည် episode card များ၏ ဘေးတိုက်
  row ဖြစ်သည် (Netflix ပုံစံ ဘေးတိုက် scroll)။ Card တစ်ခုစီတွင်
  episode thumbnail, `SxxExx`, ခေါင်းစဉ်, runtime/rating,
  **○ / ✓ Mark-watched** toggle နှင့် hover တွင် **▶ Play**။

### Episode ကြည့်ခြင်း

- Episode card ကို **နှစ်ချက်နှိပ်** (သို့) **▶ Play** နှိပ်လျှင် default
  video player ဖြင့် ဖွင့်သည်။ ဖွင့်လျှင် watched အဖြစ်လည်း မှတ်ပေးသည်။
- Card ၏ **○ / ✓** toggle က မဖွင့်ဘဲ watched state ပြောင်းပါသည်။
- Show ၏   progress roll-up ချက်ချင်း update ဖြစ်သည်။

### Show အတွက် Favorite / Watchlist

Favorite နှင့် Watchlist က **Show-level** ဖြစ်သည် (episode တစ်ခုမဟုတ်)၊
Show header ရှိ ခလုတ်များမှ ပြုလုပ်နိုင်ပါသည်။ Watched က **episode တစ်ခုစီ**။ အတွက်သာ ဖြစ်ပါသည်။ 

### Show များလည်း Drive နဲ့ အတူ သယ်သွားသည်

ရုပ်ရှင်များကဲ့သို့ပင်၊ Show ၏ personal state (favorite / watchlist /
 / episode တစ်ခုစီ watched) ကို show folder ထဲ
`cinelibrary-state.json` တွင် မှတ်ထားပေးပါသည်။ Drive ဖျက်ပြီး ပြန်ထည့်လျှင်
progress ပြန်ပေါ်လာပါမည်။

## My Lists — ကိုယ်ပိုင် List များ ဖန်တီးခြင်း

Favorites / Watchlist က ဖွင့်/ပိတ်ပဲ ရှိသည်။ 
"မိုးအေးအေးမှာ ကြည့်ရန် Drama ကားများ"၊
"ပြန်ကြည့်ရန်"၊ 
"ငါ အော်စကာပေးချင်တဲ့ ဓါတ်ရှင်များ" စသဖြင့် ကိုယ်ပိုင်
List များ ဖန်တီးချင်လာသောအခါ Sidebar ၏ **MY LISTS** အောက်တွင် ဖန်တီးနိုင်ပါသည်။

### List အသစ်တစ်ခု ဖန်တီးခြင်း

ရုပ်ရှင် poster ပေါ်တွင် right-click (သို့မဟုတ် detail dialog ဖွင့်ပြီး)
**📑 Add to list → + New list…** ကို ရွေး၍ နာမည်ထည့်ပြီး **Create**
နှိပ်ပါ။ ထို ရုပ်ရှင်သည် List အသစ်ထဲ အလိုအလျောက် ဝင်သွားလိမ့်မည်။

### ရုပ်ရှင်တစ်ခုကို List တစ်ခု (သို့မဟုတ် များစွာ) သို့ ထည့်/ဖြုတ်ခြင်း

Poster ပေါ်တွင် right-click → **📑 Add to list** ဖြင့် စာရင်းအားလုံး
ပြသပါမည်။ ရှိနေပြီးသား စာရင်းတွင် ✓ မှတ်ထားသည်။ စာရင်းတစ်ခုကို နှိပ်လျှင်
ထည့်သည်၊ ✓ နှင့်စာရင်းကို ပြန်နှိပ်လျှင် ဖြုတ်ပစ်သည်။

Detail dialog ထဲတွင်လည်း Action button များအောက်တွင် **📑 chip များ**
တန်းတစ်ခု ပေါ်လာသည် — ထို ရုပ်ရှင်ပါသော စာရင်းတိုင်းအတွက် chip တစ်ခုစီ။
Chip ၏ **✕** ကို နှိပ်လျှင် ထို စာရင်းမှ ချက်ချင်း ဖြုတ်ပစ်ပါသည်။

### List ကို Rename / Delete ပြုလုပ်ခြင်း

Sidebar ထဲ စာရင်းနာမည်ပေါ်တွင် right-click —

- **Rename** — နာမည်အသစ်ထည့်ပါ။ စာရင်းအတွင်းရှိ ရုပ်ရှင်များ မထိခိုက်ပါ။
- **Delete list** — စာရင်းကို ဖျက်သည်။ ရုပ်ရှင်များ ဘယ်တော့မှ မဖျက်ပါ —
  စာရင်းမှသာ ထွက်ပါသည်။
- **📂 Copy movies to folder…** — စာရင်းတွင်းရှိ online ရုပ်ရှင်များ၏
  source ဖိုင်တွဲကို ရွေးခြယ်ထားသော folder သို့ ကူးခြင်း (သူငယ်ချင်းကို
  USB ဖြင့် ကူးပေးချင်လျှင် အသုံးဝင်သည်)။

### List ကို ကြည့်ရှု/ပြုပြင်ခြင်း

Sidebar ထဲ စာရင်းကို နှိပ်လျှင် ၎င်းတွင်ပါသော ရုပ်ရှင်များသာ ပြသပါမည်။
ထို view ထဲမှ —

- Poster ပေါ်တွင် right-click → **📑 Add to list** → ✓ ပါသော လက်ရှိစာရင်းကို
  ပြန်နှိပ်ပြီး ဖြုတ်နိုင်သည်။
- Multi-select ဖြင့် တစ်ပြိုင်နက် အများကြီး ဖြုတ်နိုင်သည် (အောက်တွင် ကြည့်ပါ)။

## Multi-select — တစ်ပြိုင်နက် ရွေး၍ တစ်ပြိုင်နက် ဆောင်ရွက်ခြင်း

ရုပ်ရှင် ၁၂ ကား watched မှတ်ခြင်း၊ ဒါရိုက်တာတစ်ဦး၏ movie အားလုံးကို
စာရင်းတစ်ခုသို့ ထည့်ခြင်း ပုံစံများကို တစ်ခုစီ နှိပ်ရတာ ပင်ပန်းသည်။ CineLibrary
၏ Multi-select က ၎င်းကို ဖြေရှင်းပေးသည်။

### Card များ ရွေးခြယ်ခြင်း

- **Ctrl+click** — card ကို selection ထဲ ထည့်/ဖြုတ်။ ရွေးထားသော card များတွင်
  ခရမ်းရောင်ဘောင် + ထောင့်တွင် ✓ ပြသည်။
- **Shift+click** — နောက်ဆုံးနှိပ်ထားသော card မှ ထို card အထိ အကွာအဝေး ရွေးခြယ်။
- **Ctrl+A** — လက်ရှိမြင်နေသော card အားလုံး ရွေး။
- **Esc** — ရွေးထားသည်ကို ဖျက်ထုတ်။
- Selection ရှိနေစဉ် ပုံမှန် click — selection ကိုသာ ဖျက်ထုတ်သည် (detail
  မဖွင့်ပါ)။ Selection မရှိလျှင် ပုံမှန် click က detail ဖွင့်သည်။

Card တစ်ခု သို့ ထို့ထက်များသော card ရွေးထားလျှင် library ၏ အောက်ဘက်တွင်
ခရမ်းရောင် pill bar တစ်ခု ပေါ်လာမည် —

- **📑 Add to list ▾** — စာရင်းတစ်ခု ရွေးပါ။ ရွေးထားသော ရုပ်ရှင် **အားလုံး**
  ထို စာရင်းသို့ ထည့်ပါမည် (+ New list… လည်း ပါသည်)။

  အားလုံးကို ပိတ်ပေးသည်။
- **🗑 Remove from list** — လက်ရှိ မိမိစာရင်းတစ်ခုကို ကြည့်နေချိန်တွင်သာ
  ပေါ်လာသည်။ ရွေးထားသော ရုပ်ရှင်များကို ထို စာရင်းမှသာ ဖြုတ်ပစ်သည် (Library
  ထဲမှ မဖျက်ပါ၊ အခြားစာရင်းများတွင်လည်း ရှိနေဆဲ ဖြစ်သည်)။
- Action တိုင်းတွင် **Undo** ခလုတ်ပါသော toast ၆ စက္ကန့်ပေါ်နေသည် — အလွဲ
  လုပ်မိလျှင် Undo ဖြင့် ပြန် ဆယ်နိုင်သည်။

- **My List** ပေါ်တွင် drop → ရွေးထားသော ရုပ်ရှင်အားလုံး စာရင်းသို့ ထည့်။

## Drive တွေနဲ့ အတူ သယ်သွားနိုင်တဲ့ Personal Data

**v2.7.0** မှစတင်၍ — Watched / Favorite / Watchlist / List / Notes
များကို **CineLibrary ထဲမှာ ရော Movie folder ထဲမှာ ပါ** သိမ်းပါမည်။
Drive ကို CineLibrary မှ ဖျက်လိုက်ပြီး နောက်တစ်ကြိမ် ပြန် Add လျှင်လည်း 
အရင်မှတ်သားထားခဲ့သော Mark တွေ ပြန်ပေါ်လာပါမည်။

### ဘယ်လို အလုပ်လုပ်သလဲ?

Personal data (Watched / Favorite / Watchlist / Played history /
ပါဝင်နေသော List / Note) ရှိသော ရုပ်ရှင် တစ်ကားစီ၏ folder ထဲတွင်
**`cinelibrary-state.json`** ဆိုသော ဖိုင်ငယ်တစ်ခုကို ရေးပါသည် —
`.nfo` ဖိုင်နှင့် တွဲထားပါသည်။ အပြင်အဆင် ဒီပုံစံ —

```json
{
  "version": 1,
  "watched": true,
  "favorite": true,
  "lastPlayedUnix": 1747576800,
  "lists": ["အကောင်းဆုံး ၁၀ ကား"]
}
```

တချက်ခုတ် သုံးချက်ပြတ် အလိုအလျောက် စနစ် သုံးထားပါသည် —

- **Mark / Unmark / Play / List / Note ပြောင်းတိုင်း** 
  ဖိုင်ကို CineLibrary ထဲ  ရေးမှတ်ထားပေးပါသည်။ Best-effort — Drive offline ဖြစ်နေလျှင် Drive ပလပ်ထိုးချိန် CineLibrary ထဲမှ ဒေတာကို Drive ထဲ အလိုအလျောက် sync လုပ်ပါလိမ့်မည်။
- **Drive ကို scan / rescan လုပ်တိုင်း**၊ `cinelibrary-state.json`
  ဖိုင်တွေ့လျှင် ဖတ်၍ Library သို့ ပြန်ထည့်ပါသည်။ 
  "Drive ဖျက်ပြီး  ပြန် Add" သော် Library ထဲ မှတ်သားထားခဲ့သမျှ အားလုံး ပြန်ပါလာပါမည်။
- **App ကိုဖွင့်လိုက်တိုင်း**၊ background မှ လက်ရှိ personal state များ ဖိုင်တွေထဲ ပါပြီးပြီလားဆိုတာကို အလိုအလျောက် ရှာဖွေ ဖြည့်စွက်ပေးပါမည်။

### Remove Drive Dialog

Drives စာမျက်နှာတွင် 🗑 Remove Drive နှိပ်လျှင်၊ ထို drive ပေါ်တွင်
personal data ပါသော ရုပ်ရှင် မည်မျှရှိသည်ကို ပြပြီး၊ **"Drive
ထဲ ဦးစွာ သိမ်းပါ"** checkbox (default ON) ပေါ်လာပါမည်။ ထို checkbox
အွန်ထားလျှင် Drive ကို နောက်တစ်ကြိမ် ပြန်ထည့်သောအခါ အရင်က Personal
Data များ အားလုံး ပြန်ပေါ်လာပါမည်။

Drive **offline** ဖြစ်နေချိန်တွင် ဖျက်ဖို့ ကြိုးစားလျှင်၊ Dialog က
Personal Data များ မသိမ်းနိုင်ကြောင်း သတိပေး၍ Drive ပြန်ချိတ်ပြီးမှ
ဆက်လုပ်ရန် အကြံပြုပါမည်။

### Sync state ခလုတ်

Drive card တိုင်းတွင် **Sync state** ခလုတ်တစ်ခုစီ ထည့်ပေးထားပါသည်။ ပုံမှန်
အသုံးပြုသူများအတွက် မလိုပါ — အလိုအလျောက် sync က နေ့စဉ်ဆောင်ရွက်၍
အလိုလို ဖြေရှင်းပေးပြီးသားဖြစ်ပါသည်။ ဤခလုတ်က ဘယ်အချိန် အသုံးဝင်သလဲဆိုတော့ —

- ဖိုင်တွေ disk ထဲ ချက်ချင်း ထည့်ချင်သည် (ဥပမာ Drive ကို
  သူငယ်ချင်းကို လွှဲချင်သည်)။
- `CineLibrary-Data\cinelibrary.db` ကို backup မှ restore ပြုလုပ်
  ပြီးနောက် Drive တွေနဲ့ ပြန် sync ချင်သည်။
- Drive offline ဖြစ်နေစဉ်တွင် ပြုပြင်ပြောင်းလဲမှု များစွာရှိခဲ့ပြီး၊
  မှတ်သားခဲ့သမျှအားလုံးကို  catch-up ဖြစ်စေချင်သည့်အချိန်။

### ပဋိပက္ခ ဖြစ်ပွားမှုဆိုင်ရာ စည်းမျဉ်း
သင့်မှာ တန်ဖိုးမတူညီတဲ့ DB row တစ်ခုနဲ့ sidecar ဖိုင်တစ်ခု ရှိနေပါက 
(ဥပမာ: သင်က စက် A မှာ တစ်ခုခုကို ကြည့်ပြီးကြောင်း မှတ်သားခဲ့ပြီး၊ 
အဲဒီနောက် ၎င်း drive ကို မိတ္တူရှိနှင့်ပြီးသားဖြစ်တဲ့ စက် B ထဲကို သွားစိုက်မိတဲ့အခါမျိုး)၊ 
တန်ဖိုး ရှိနှင့်ပြီးသားဖြစ်တဲ့ DB ဘက်ကပဲ အတည်ဖြစ်မှာ ဖြစ်ပါတယ်။ 
စာရင်း (Lists) တွေကိုတော့ ပေါင်းစပ်ပေးသွားမှာ ဖြစ်ပါတယ်: 
စာရင်းထဲမရှိသေးတာကို ပေါင်းထည့်ပေးမှာဖြစ်ပြီး၊ မည်သည့်အရာကိုမျှ အလိုအလျောက် တိတ်တဆိတ် ဖျက်ထုတ်ပစ်မှာ မဟုတ်ပါဘူး။

### သက်ရောက်မှုမရှိသည့် အစိတ်အပိုင်းများ
cinelibrary-state.json ဟာ CineLibrary က သင့်ရဲ့ ရုပ်ရှင်ဖိုဒါ (movie folder) ထဲကို ရေးသားတဲ့ တစ်ခုတည်းသော ဖိုင်ဖြစ်ပါတယ်။ 
သင့်ရဲ့ .nfo၊ ပိုစတာတွေ၊ fanart တွေနဲ့ ဗီဒီယိုဖိုင်ကိုယ်တိုင်တို့ကိုတော့ ဘယ်တော့မှ ပြုပြင်ပြောင်းလဲမှာ မဟုတ်ပါဘူး။ 
sidecar ဖိုင်ဟာ ရုပ်ရှင်တစ်ကားချင်းစီအတွက် byte ရာဂဏန်းအနည်းငယ်ပဲ ရှိပြီး ဖျက်ပစ်လိုက်လည်း စိတ်ချရပါတယ်။ 
နောက်တစ်ကြိမ် အပြောင်းအလဲတစ်ခုခု ရှိလာတာနဲ့ CineLibrary က ၎င်းဖိုင်ကို ပြန်လည်ဖန်တီးပေးသွားမှာ ဖြစ်ပါတယ်။

## Drive များ — အွန်လိုင်းနှင့် အော့ဖ်လိုင်း

CineLibrary ကို External drive များစွာ တစ်ပြိုင်နက် မချိတ်ဆက်ထားသော အခြေအနေကို ထည့်တွက်၍ ဒီဇိုင်းဆွဲထားပါသည်။

- Drive **ချိတ်ဆက်ထားလျှင်** sidebar တွင် အစိမ်းရောင် dot ပြပါမည်။
- Drive **ဖြုတ်ထားလျှင်** dot မှိန်သောအရောင် ဖြစ်သည်၊ ရုပ်ရှင်များ grid တွင် `OFFLINE` overlay ဖြင့် ပြသည်။

Drive ချိတ်ဆက်ချိန်နှင့် ဖြုတ်ချိန် Windows device event ဖြင့် **ချက်ချင်း** သိရှိပါလိမ့်မည် — polling မရှိ၊ နှေးကွေးမှု မရှိ။ Drive ချိတ်ချိန် တစ်စက္ကန့်အတွင်း ရုပ်ရှင်များ ဖွင့်ကြည့်၍ ရလာပါမည်။

## Catalog ထုတ်ယူခြင်း

ထိပ်ဆုံး **Export** button နှိပ်၍ လက်ရှိ view ကို ထုတ်ယူနိုင်သည် — filter နှင့် sort အားလုံး ပါဝင်သည်။ ပုံစံ နှစ်မျိုး —

- **CSV** — Excel / Numbers တွင် ဖွင့်နိုင်သော spreadsheet ပုံစံ၊ ရုပ်ရှင်တစ်ကားစီ တစ်ကြောင်းတည်း၊ ခေါင်းစဉ်၊ နှစ်၊ runtime၊ rating၊ genre၊ drive၊ ကြည့်ပြီးမပြီး၊ ဖိုင်လမ်းကြောင်း ပါသည်။
- **HTML** — Poster thumbnail များ ပါသော self-contained HTML ဖိုင်တစ်ဖိုင်။ CineLibrary မရှိသော သူများနှင့် catalog မျှဝေရာတွင် အသုံးဝင်သည်။

Export သည် လက်ရှိ filter ကို လိုက်နာသည် — Watchlist မှသာ ထုတ်ယူလိုလျှင် *Watchlist* ကို filter ပြုလုပ်ပြီးမှ Export နှိပ်ပါ။

## ဒေတာ သိမ်းဆည်းနေသောနေရာ

CineLibrary ၏ catalog ဒေတာအားလုံး exe ဘေးရှိ `CineLibrary-Data\` တွင် ထားရှိပါသည် —

```
<install folder>\
├── CineLibrary.exe
├── CineLibrary-Data\
│   ├── cinelibrary.db        # SQLite catalog (ရုပ်ရှင်၊ drive၊ settings)
│   └── cache\                # Poster/thumbnail cache (compressed)
└── ...
```

ဤဖိုင်တွဲသည် **portable** ဖြစ်သည်။ install folder တစ်ခုလုံးကို အခြားကွန်ပျူတာတစ်လုံး၊ USB stick သို့မဟုတ် external drive ထဲသို့ ကူးယူသယ်ဆောင်သွားနိုင်ပါသည် — app နှင့် catalog အတူပါသည်။ **Telemetry မရှိ**၊ cloud sync မရှိ၊ account မလိုအပ်ပါ။ Library ဒေတာ သင့်ကွန်ပျူတာမှ မထွက်ပါ။

## ပြဿနာဖြေရှင်းခြင်း

### ထည့်သွင်းသော ရုပ်ရှင် မပေါ်ဘူး

ထို drive ပေါ်တွင် **Rescan** နှိပ်ပါ။ CineLibrary သည် `.nfo` ဖိုင်ရှိသော ဖိုင်တွဲများကိုသာ catalog ပြုလုပ်သည်၊ MediaElch ဖြင့် မ scrape ရသေးလျှင် မပေါ်ပါ။

### Poster ပုံများ မပေါ်ဘူး သို့မဟုတ် မှားနေတယ်

CineLibrary သည် MediaElch ၏ convention အတိုင်း `poster.jpg` (သို့မဟုတ် `folder.jpg`၊ `.nfo` နှင့်အတူ ရှိသော အကြီးဆုံးပုံ) ဖတ်သည်။ MediaElch ဖြင့် ပြန် scrape ပြုလုပ်ပြီး CineLibrary တွင် ထို drive ကို ပြန် rescan ပါ။

### Drive ချိတ်ထားသော်လည်း OFFLINE ပြနေတယ်

**Drives** စာမျက်နှာ ဖွင့်၍ connection dot စစ်ပါ။ Drive ချိတ်ထားသော်လည်း dot မှိန်နေလျှင် drive ၏ volume serial ပြောင်းသွားနိုင်သည် (reformat သို့မဟုတ် partition ပြောင်းပြီးနောက် ဖြစ်တတ်သည်)။ Drive entry ဖျက်ပြီး root ဖိုင်တွဲကို ပြန်ထည့်ပါ — CineLibrary ၏ catalog ချိန်ညှိသွားမည်။

### CineLibrary မဖွင့်နိုင်ဘူး / ချက်ချင်း ပိတ်သွားတယ်

- Windows 10 build 19041 သို့မဟုတ် အထက် ဖြစ်မဖြစ် စစ်ပါ (`winver`)။
- Install ဖိုင်တွဲ network share (UNC path) ပေါ်တွင် မဟုတ်ကြောင်း သေချာပါစေ — WinUI တွင် UNC path ၌ ပြဿနာများ ဖြစ်တတ်သည်။
- ပြဿနာ ဆက်ဖြစ်နေလျှင် [GitHub](https://github.com/aungkokomm/CineLibraryCS/issues) တွင် issue တင်ပါ — launcher script ၏ exit code နှင့် ဖြစ်ရပ်ကို အကျဉ်းချုပ် ဖော်ပြပါ။ ကိုယ်ရေးကိုယ်တာ path များ ခေါင်းစဉ်တွင် မထည့်ပါနှင့်။

### Catalog အသစ် ပြန်စချင်တယ်

CineLibrary ပိတ်ပါ။ Exe ဘေးရှိ `CineLibrary-Data\` ဖိုင်တွဲ ဖျက်ပါ။ ပြန်ဖွင့်ပါ — CineLibrary သည် database အသစ် ဖန်တီးမည်။ ရုပ်ရှင်ဖိုင်များ မထိခိုက်ပါ။
